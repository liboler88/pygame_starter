# main1.py 程式說明檔

這是一個使用 Pygame 實作的基礎貪吃蛇控制原型。

## 🎯 程式目標
這個腳本的主要目的是展示如何：
1.  開啟一個 Pygame 視窗。
2.  偵測使用者的鍵盤輸入 (上下左右)。
3.  移動一個幾何圖形 (綠色方塊)。
4.  實作「穿牆」機制 (螢幕邊界循環)。

## 🎮 如何操作
執行腳本：
```bash
python main1.py
```

| 按鍵 | 動作 | 程式碼對應邏輯 |
| :--- | :--- | :--- |
| **↑** | 往上 | `direction = [0, -5]` |
| **↓** | 往下 | `direction = [0, 5]` |
| **←** | 往左 | `direction = [-5, 0]` |
| **→** | 往右 | `direction = [5, 0]` |

## 🛠️ 程式核心結構

### 1. 狀態變數
```python
snake_pos = [320, 240]       # (X, Y) 座標
direction = [snake_speed, 0] # 移動向量
```

### 2. 遊戲迴圈 (Game Loop)
程式在 `while running:` 迴圈中不斷執行以下三件事：
1.  **事件處理**：監聽 `KEYDOWN` 事件來更新 `direction`。
2.  **更新邏輯**：將 `direction` 加到 `snake_pos` 上。若超出邊界 (0~640, 0~480) 則重設位置。
3.  **繪圖渲染**：使用 `screen.fill("black")` 清除畫面，再畫上綠色方塊。
